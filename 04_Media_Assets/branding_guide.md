# Branding & Visual Identity Guide: REFUEL

## 1. Core Visual Aesthetic (Horology/Breguet)
- **Primary Palette:** #FFFDD0 (Cream/Milk), #7097BB (Slate Blue).
- **Accents:** #B8D8BA (Sage Green), #F7CFD1 (Champagne Pink).
- **Style:** Breguet hands (Navy Blue), Roman numerals (Serif), and Guilloché patterns.

## 2. Active Mode (Digital HUD)
- **Satoshi Mode:** International Orange (#F7931A) on Charcoal Grey (#121212).
- **Cashback Mode:** Electric Cobalt (#2E5BFF) on Charcoal Grey (#121212).
- **Style:** Glassmorphism and SF Pro (Sans-serif) for data.

## 3. High-Fidelity State Variations
- **The "Station Master" Flip (B2B Host):**
    - **Palette:** Deep Purple (#4B0082) with Electric Orange accents.
    - **Key Visual:** "Idle RPM" ticker (Real-time earnings/throughput per hour).
- **The "Queue Buster" (Costco Infrastructure):**
    - **Component:** Dynamic Island / Live Activity.
    - **Visual:** "Next Pump: 45s" countdown timer with high-contrast text.
- **The "Retail Valet" (Lakeside Context):**
    - **Watch Variant:** Replace central 'Speedometer' with a 'Shopping Timer' ring.

## 4. Mode-Specific Visual Language
- **Resident Mode (The Commuter):** 
    - **Color:** Electric Green (#00FF00) - Symbolizes growth and savings.
    - **Primary Component:** 'Weekly Progress Ring' toward Satoshi/Coffee rewards.
- **Diplomat Mode (The Traveler):** 
    - **Color:** International Orange (#F7931A) - Symbolizes caution/alerts and Bitcoin.
    - **Primary Component:** 'Distance-to-Airport' radial dial.
- **Station Master Mode (The Host):** 
    - **Color:** Cyber Purple (#8A2BE2) - Symbolizes 'The Power Plant' and high-yield arbitrage.
    - **Primary Component:** 'Guest SoC' (State of Charge) ring.
