# REFUELER | Command Centre
**The Operational Interface for the Refueler Protocol.**

## Executive Summary
The Command Centre is a high-fidelity, dual-mode dashboard designed for infrastructure management (Audit) and retail settlement (Retail). It integrates the Lightning Development Kit (LDK) with geospatial site-auditing to verify and reward user proximity in real-time.

## Key Technologies
* **Protocol:** LDK (Lightning), Cashu/Minibits (eCash), Fountain (Podcasting 2.0).
* **Geospatial:** Mapbox GL JS with custom dark-v11/light-v11 styling.
* **Infrastructure:** Open Payments Cloud (OPC) for CarPlay & Vehicle integration.
* **Frontend:** Vanilla JS / CSS3 (Hairline Aesthetic System).

## Core Features
* **Geofence Verification:** Real-time LDK session handshake via the "Verify Dial."
* **Dual-Mode Toggle:** Seamless transition between technical Site Audits and Retail ROI views.
* **Telemetry Stream:** Low-latency system logs for transaction transparency.
* **Multi-Platform:** Designed for desktop command, merchant handhelds, and CarPlay head units.

## Design Philosophy
The UI follows a "Carbon-First" aesthetic, prioritizing high contrast and legibility. It utilizes 0.5px hairline dividers to mirror high-end automotive and financial software interfaces.
