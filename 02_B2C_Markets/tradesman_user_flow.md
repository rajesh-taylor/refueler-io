# User Flow: Mode C (The Tradesman)

## Scenario: High-Roof Transit (High Stakes/Zero Admin)
**Objective:** Navigate to a compatible station and automate business compliance.

### Stage 1: The Rig Setup
- **Action:** User enters Reg "VN62 XRA".
- **Refueler Logic:** Fetch 2.6m height threshold.
- **Visual UI:** 3D wireframe van confirms "High-Roof Safety Active."

### Stage 2: The Clearance Filter

- **CarPlay UI:** Rejects Shell (2.4m canopy). Routes to BP (2.8m clearance).
- **Voice Nudge:** "Diverting to BP. Avoided low canopy at Shell. Saved 15 mins."

### Stage 3: The Invisible Admin
- **Action:** Nozzle holstered.
- **Watch UI:** Triple-Pulse Haptic. Displays: "£85.00 Paid | £14.16 VAT Secured."
- **Action:** Digital invoice injected into Xero API background thread.

### Stage 4: The Glovebox Audit
- **Action:** Evening review of "Recent Fills."
- **UI:** Green "Clean" checkmarks for matched transactions. 
- **OCR Fallback:** Use "Live Camera" to scan paper receipt for ⚠️ flagged items (e.g., meal deal split).
